= gomasaba

gomasaba は、実験と Zig 学習のために実装している、WebAssembly ランタイムです。

== ビルド

zig 0.10.1 をインストールし、以下を実行します。

[source, console]
----
zig build -Drelease-safe
----

`./zig-out/bin/gomasaba` が出力されます。
